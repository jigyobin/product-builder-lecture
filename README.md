# Lotto Number Generator

A simple lotto number generator built with HTML, CSS, and JavaScript.

## Features
- Random 6-number generation (1-45)
- Dark/Light mode support (persists in localStorage)
- Responsive design

## Deployment
- Hosted on GitHub Pages: [https://jigyobin.github.io/product-builder-lecture/](https://jigyobin.github.io/product-builder-lecture/)
- Connected to Cloudflare Pages for automatic deployment.
